Liaison entre les pages.

Toutes les pages sont liées à la page d'accueil. Ils ont pour extension .html[...]. cela permet au navigateur de comprendre qu'il s'agit d'un code html; non seulement ça, il comprend que nous souhaitons passer d'une page à l'autre. Un click suffit d'accéder à une autre page html ou page du site web. La balise utilisée pour les liens est la balise (a) délimitée par les chevrons entre crochet {< >}.

Retour à la page d'accueil.

Le titre de la page contient un lien du code source de la page. C'est-à-dire, lien du code source de la page html exemple: index.html. Juste un clique sur le titre et vous vous retrouvez à la page d'accueil.

Code source.

Le code source de la page est essentiellement html et css. Html est un langage basic sans style, c'est-à-dire ne permet pas de donner un style de font (couleur), structure de la page et autres. Mais il contient toutes les balises permettant de faire une page web.
En effet, le CSS est un langage de style composé des balises de style pour structurer une page web et lui donner un fond, des animations et plus. De ce fait, il n'y a pas de javaScript, ni PHP,...
